# Contributing to AGENSTAB

Thank you for your interest in contributing to AGENSTAB.

## How to Contribute

### Bug Reports
- Open an issue with a clear description, steps to reproduce, and expected vs actual behavior
- Include your SDK version and language

### Feature Requests
- Open an issue tagged `[Feature Request]`
- Describe the use case, not just the solution

### Code Contributions
1. Fork this repository
2. Create a feature branch (`git checkout -b feature/your-feature`)
3. Commit your changes (`git commit -m 'Add your feature'`)
4. Push to the branch (`git push origin feature/your-feature`)
5. Open a Pull Request

### SDK Examples
We especially welcome example scripts showing AGENSTAB used with:
- LangChain / LangGraph
- AutoGPT / CrewAI
- Custom agent frameworks

### Documentation
Improvements to docs, typo fixes, and tutorial additions are always welcome.

## Code Style
- Python: Follow PEP 8
- JavaScript: ESLint with standard config
- Go: `gofmt`
